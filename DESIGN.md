# 语音绘图工具 — 设计文档

## 1. 项目概述

纯语音控制的绘图工具，用户仅通过中文语音指令在 800×600 画布上创作图形。核心目标：**高准确率的语音理解 + 低延迟响应 + 丰富的指令覆盖**。

### 技术选型

| 组件 | 选择 | 理由 |
|------|------|------|
| 语音识别 | faster-whisper (small, int8) | 纯离线、中文精度高、CPU 可运行 |
| 拼音纠错 | pypinyin + 编辑距离 | 解决同音字问题（如"元"→"圆"） |
| 命令解析 | 正则 + 模糊匹配 + 场景分解 | 轻量、可预测、适合有限指令集 |
| 画布渲染 | OpenCV + PIL | 内置窗口系统、高性能、中文文本支持 |
| 语音反馈 | pyttsx3 | 纯离线 TTS、低延迟、无需网络 |

## 3. 语音识别架构

### 3.1 识别流水线

```
麦克风录音 (5s 超时, 6s 最大短语)
    ↓
faster-whisper 转写 (small 模型, int8, beam_size=5, best_of=3)
    ↓
置信度过滤 (avg_logprob < -1.5 丢弃, no_speech_prob > 0.8 丢弃)
    ↓
去重去噪 (_dedup_whisper_text: 重复短语去除 + 常见错误修正)
    ↓
垃圾过滤 (_is_garbage: 过短/重复字符/常见填充词)
    ↓
选择最佳候选 (_pick_best_transcript: 第一个可解析的候选)
    ↓
执行命令
```

### 3.2 容错机制

**层级 1 — 语音纠错** (`fix_speech_text`)：
- 150+ 常见 Whisper 误识别映射（同音字、近音字）
- 去除语气词（嗯、啊、呃）
- 去除重复字符
- 去除礼貌前缀（帮我、给我、请）
- 颜色+形状间"的"字去除（红色的圆 → 红圆）

**层级 2 — 中英翻译** (`zh_to_en`)：
- 150+ 中文→英文映射，按长度优先匹配
- 中文数字转换（一百 → 100）

**层级 3 — 正则匹配** (`_parse_single`)：
- 32 个正则模式，按注册顺序尝试

**层级 4 — 拼音模糊匹配** (`_pinyin_fallback`)：
- 将中文转为拼音，与 150+ 已知命令比较编辑距离
- 最大允许 2 处拼音差异
- 解决："开画"(kaihua) → 匹配 "开花"(kaihua)

**层级 5 — 关键词模糊匹配** (`_fuzzy_fallback`)：
- difflib 颜色匹配 (cutoff=0.68)
- 裸关键词检测（无动词也能触发）
- 场景模板匹配

### 3.3 延迟优化

| 优化项 | 措施 |
|--------|------|
| Whisper 参数 | beam_size=5, best_of=3（平衡精度与速度） |
| 量化 | int8 计算，减少 CPU 内存占用 |
| VAD 过滤 | 自动检测语音段，跳过静音 |
| 异步保存 | 自动保存 PNG 在独立线程执行 |
| 语音反馈 | pyttsx3 在独立线程播报，不阻塞主循环 |

## 5. 创新点

1. **多层级容错**：语音纠错 → 拼音模糊 → 关键词匹配，5 层递进式容错
2. **纯离线方案**：faster-whisper + pyttsx3，无需网络，保护隐私
3. **场景描述引擎**：自然语言自动布局，"画一幅画：左边树右边山"一次完成
4. **拼音纠错**：利用中文同音字特性，解决语音识别的根本性难题
5. **语音确认**：二次确认通过语音触发（再说一次"清空"），而非键盘
